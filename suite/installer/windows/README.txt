                          Getting SeaMonkey

==========================================================================

You can download nightly builds of SeaMonkey from the mozilla.org FTP site
at

  http://ftp.mozilla.org/pub/mozilla.org/seamonkey/nightly/

For the very latest builds, see

  http://ftp.mozilla.org/pub/mozilla.org/seamonkey/nightly/latest-trunk/

Keep in mind that nightly builds, which are used by mozilla.org developers
for testing, may be buggy. If you are looking for a more polished version
of SeaMonkey, the SeaMonkey project releases builds of SeaMonkey regularly
that you can download from

  http://www.seamonkey-project.org/releases/

Be sure to read the SeaMonkey release notes for information on known
problems and installation issues with SeaMonkey.  The release notes can be
found at the preceding URL along with the releases themselves.


==========================================================================

                         System Requirements

==========================================================================

*Windows

   - Windows 2000, XP ou Vista
   - Processador Intel Pentium (recomendado 500 MHz ou mais rápido)
   - 128 MB RAM
   - 50 MB de espaço livre em disco


==========================================================================

                      Instruções de instalação

==========================================================================

É recomendado que saia de todos os programas antes de correr o programa
de instalação. Também é aconselhado desactivar o seu software de anti-virús.

Instale numa nova directoria.
Instalar em cima de versões anteriores pode causar problemas.

Nota: Estas instruções não lhe dizem como fazer o SeaMonkey.
Para info sobre como fazer o SeaMonkey a partir do código fonte da 
mozilla.org, veja

  http://developer.mozilla.org/en/docs/Build_Documentation


Instruções de instalação para Windows
-------------------------------------

Nota: Para sistemas Windows 2000/XP, necessita de previlégos de Administrador 
para instalar o SeaMonkey. Se vir uma mensagem "Erro 5" durante a instalação,
verifique se está a instalar com previlégios de administrador.


   Para instalar o SeaMonkey com o programa de instalação,
   siga os seguintes passos:

   1. Clique na ligação do seamonkey-x.xx.en-US.win32.installer.exe (ou
      ficheiro com o nome parecido) no sítio de onde vai transferir o SeaMonkey
      para guardar o ficheiro no seu computador.
      
   2. Vá até onde guardou o ficheiro de instalação e clique duas vezes no 
      ficheiro de instalação para iniciar o programa de instalação.

   3. Siga as instruções que aparecem no ecrã. O programa inicia automáticamente.


   Para instalar o SeaMonkey com o ficheiro .zip e instalar manualmente
   siga os seguintes passos:

   1. Clique na ligação do seamonkey-x.xx.en-US.win32.zip (ou
      ficheiro com o nome parecido) no sítio de onde vai transferir o SeaMonkey
      para guardar o ficheiro .zip no seu computador.

   2. Vá até onde guardou o ficheiro e clique duas vezes no ficheiro comprimido

      Nota: Este passo assume que tem uma versão recente do WinZip (ou outra 
      aplicação similar) instalada, e que sabe usá-la.
      Se não tiver, pode obtr o WinZip e informação sobre o programa em
      www.winzip.com.

   3. Extraia o ficheiro .zip file para uma directoria tipo
        C:\Programas\mozilla.org\SeaMonkey.

   4. Para iniciar o SeaMonkey, vá até à directoria para onde extraiu o
      SeaMonkey e clique duas vezes no ícone seamonkey.exe.
