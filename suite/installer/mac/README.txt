  Requisitos de Sistema & Hardware
  --------------------------------

  Requisitos de Sistema:

  - Mac OS X 10.4 ou mais recente


  Requisitos de Hardware:

  - Computador Macintosh com um processador Intel x86 ou PowerPC G3/G4/G5
  - 256 MB RAM
  - 200 MB de espaço livre no disco


  Instalação
  ------------

  Para instalar o SeaMonkey, siga os seguintes passos:

  1. Arraste o ícone da aplicação SeaMonkey para a pasta de aplicações

  2. Ejecte a imagem do disco

  3. Se desejar que o SeaMonkey esteja acessível a partir da Dock, arraste o ícone do SeaMonkey
     da pasta de aplicações para a Dock


  Onde é que a informação é guardada
  ---------------------------

  Quando executa o SeaMonkey pela primeira vez, é criada a pasta "SeaMonkey" em

  ~/Library/Application Support/ (onde ~ é a sua pasta principal).

  As suas definições pessoais, marcadores, correio electrónico, senhas etc. são guardadas numa
  numa pasta de perfil dentro da pasta SeaMonkey. A pasta de perfil está localizada em

  ~/Library/Application Support/SeaMonkey/Profiles/[randomString.profileName].

  O SeaMonkey guarda a cache em ~/Library/Cache/SeaMonkey. Certas definições
  (Como a última pasta utilizada pelo diálogo guardar) é guardada em

  ~/Library/Preferences/org.mozilla.seamonkey.plist.


  Desinstalação
  -------------

  Para desinstalar o SeaMonkey (mantendo as suas definições pessoais, marcadores etc.),
  basta arrastar o ícone do SeaMonkey da sua pasta das aplicações para o lixo.

  Para remover completamente o SeaMonkey do seu computador
  (incluindo as suas definições pessoais, marcadores, correio electrónico, senhas etc.),
  siga os seguintes passos:

  1. Arraste o ícone do SeaMonkey da sua pasta das aplicações para o lixo

  2. Arraste a pasta em ~/Library/Application Support/SeaMonkey para o lixo

  3. Arraste a pasta em ~/Library/Cache/SeaMonkey para o lixo

  4. Arraste o ficheiro ~/Library/Preferences/org.mozilla.seamonkey.plist para o lixo


  Obter o SeaMonkey
  -----------------

  Apenas para testes, as versões nocturnas do SeaMonkey são fornecidas com regularidade.
  Elas reflectem o desenvolvimento em curso e podem ter problemas.

  Versões nocturnas de várias datas estão disponíveis no sítio ftp da mozilla.org
  em http://ftp.mozilla.org/pub/mozilla.org/seamonkey/nightly/.

  Para as últimas versões, veja

  http://ftp.mozilla.org/pub/mozilla.org/seamonkey/nightly/latest-trunk/.

  Se estiver à procura de uma versão mais polida do SeaMonkey, pode transferir as versões oficiais 
  a partir de
   
  http://www.seamonkey-project.org/releases/.
