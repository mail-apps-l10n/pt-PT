Início

Por omissão, o &brandShortName; apresentará a sua página inicial, ao ser aberto. A sua página inicial pode ser um sítio que visite frequentemente, um motor de pesquisa, ou talvez a sua conta de correio electrónico. A escolha é sua. Pode alterar a sua página inicial, escolher uma página ou conjunto de páginas diferentes no início nesta secção.

Quando o &brandShortName; inicia
Por omissão, o &brandShortName; abre com a sua página inicial (listada na caixa de texto imediatamente em baixo), que corresponde à opção Mostrar a minha página inicial. Como alternativa, pode escolher uma página em branco para ser apresentada no início (eliminando o tempo de carregamento de uma página da Internet), seleccionando a opção Mostrar uma página em branco.

Ainda como alternativa, o &brandShortName; pode reabrir as janelas e separadores que tinha aberto na última sessão, remetendo-o para os sítios onde estava quando deixou de navegar na Internet. Esta pode ser uma forma útil de guardar o que estava a fazer quando fechou o &brandShortName;, talvez para instalar uma actualização do sistema operativo. Pode escolher esta opção seleccionando Mostrar as minhas janelas e separadores da última vez.

Página inicial
Se quiser definir uma página inicial quando o &brandShortName; abrir, pode especificá-la aqui. Pode também clicar num dos seguintes botões da caixa de texto para escolher uma página inicial especial:

  • Utilizar marcador define o marcador seleccionado (ou, se tiver seleccionado uma pasta, o conjunto de marcadores dessa pasta) como a sua página inicial
  • Repor valor de origem repõe a sua página inicial de origem

OpçõesPreferências de Segurança

Avisar quando os sítios tentam instalar extras
O &brandShortName; irá pedir-lhe sempre para confirmar a instalação de extras. Será aconselhável ler e concordar com a política de privacidade do fornecedor de dados seleccionado depois de optar por esta opçãopreferência.

Com o deslocamento suave, desliza para baixo suavemente, de forma em que consegue ver o que desloca. Esta acção facilita a continuação da leitura a partir de onde estava.

Poderá ser necessário descarregar um dicionário; para o fazer, clique com o botão direito do ratoprima Ctrl e clique em qualquer campo de texto, activando o verificador automático se necessário, e depois utilize o menu de Idiomas para descarregar um dicionário.

Rede

Ligação

O seu fornecedor de Internet pode requerer o uso de um proxy. Um proxy serve de intermediário entre o seu computador e a Internet. Intercepta todos os pedidos enviados para a Internet, verificando se pode satisfazê-los utilizando a memória cache. Os Proxies são utilizados para melhorar o desempenho, filtrar pedidos, e esconder o seu computador da Internet para aumentar a segurança. Os Proxies são muitas vezes parte de firewalls.

Definições da Ligação

Ligação directa à Internet
Esta é a opçãopreferência por omissão. Seleccione-a se não quer utilizar um proxy.
Detectar automaticamente as definições de proxy para esta rede
Seleccione esta se quiser que o &brandShortName; detecte automaticamente as definições do proxy para a sua rede.
Configuração manual de proxy
Seleccione esta se não tiver uma localização de proxy (URL). Pergunte ao seu administrador de sistema pelos nomes e números das portas dos servidores que correm software do proxy de cada serviço de rede e insira os dados nos campos apropriados.
URL de configuração automática de proxy
Se o seu terminal tiver um ficheiro de configuração de proxy, peça o seu URL ao administrador de sistema e submeta-o aqui. Clique em Recarregar para carregar as definições.

Cache

Normalmente, as páginas que visita são armazenadas numa pasta especial de cache para acelerar o carregamento da página da próxima vez que a visitar. Pode especificar o espaço em disco que quer disponibilizar para este efeito. Pode também limpar imediatamente o conteúdo da cache.

Utilizar até ... MB de espaço em disco para a cache
Permite-lhe especificar o tamanho máximo, em megabytes, da cache no seu computador.

Limpar agora
Limpa imediatamente o conteúdo actual da cache, libertando o espaço em disco utilizado pela cache.

Actualizações

Verificar automaticamente a existência de actualizações para:
Por omissão, o &brandShortName; verifica automaticamente a existência de actualizações disponíveis para si próprio, para extras, e para motores de pesquisa, para que tenha sempre a versão mais actualizada. Pode alterar este comportamento, desmarcando aqui as opções correspondentes.

Quando forem encontradas actualizações para o Firefox,

Perguntar o que fazer
Por omissão, as actualizações do &brandShortName; são instaladas automaticamente. Seleccione esta opçãopreferência para controlar manualmente como e quando as actualizações são instaladas.
Transferir e instalar automaticamente a actualização
Se esta opçãopreferência estiver seleccionada, as actualizações do &brandShortName; serão automaticamente descarregadas, sendo instaladas quando o &brandShortName; for reiniciado. A opçãopreferência Avisar se este procedimento desactivar algum dos meus extras permite-lhe ser avisado antes da instalação de uma actualização, que pode necessitar de desactivar um extra incompatível, por não existir nenhuma versão mais recente e compatível. Se todas as extensões e temas forem compatíveis ou se puderem ser actualizadas nesse sentido, a actualização do &brandShortName; será instalada. Depois de reiniciar, o &brandShortName; irá pedir-lhe para instalar actualizações necessárias de extras, para poder continuar a utilizá-los.

Mostrar o históricos das actualizações
O &brandShortName; regista automaticamente as actualizações instaladas. Ao clicar neste botão, terá acesso à informação sobre estas actualizações.

Nota: Para instalar as actualizações do &brandShortName;, é necessário corrê-lo como sendo administradorroot ou como o utilizador que instalou o &brandShortName; originalmente.

Cifra

Protocolos

Utilizar SSL 3.0
Especifica se quer enviar e receber informação de forma segura utilizando SSL3 (Secure Sockets Layer, Level 3), um protocolo comum para transmissões seguras em sítios web. Ao desactivar esta opção, não poderá aceder a alguns sítios.

Utilizar TLS 1.0
Especifica se quer enviar e receber informação de forma segura utilizando TLS (Transport Layer Security), uma norma de segurança semelhante ao SSL3 (Secure Sockets Layer). Ao desactivar esta opção, não poderá aceder a alguns sítios.

Certificados

Os certificados ajudam a cifrar e a decifrar ligações para sítios seguros.

Quando um sítio necessitar de um certificado:
Quando um sítio web necessita de uma ligação segura, o &brandShortName;, por omissão, utiliza um certificado apropriado automaticamente. Se quiser escolher manualmente um certificado (por exemplo, se quiser utilizar um certo tipo de cifra em vez da seleccionada automaticamente), seleccione a &pref.singular; perguntar sempre e terá controlo absoluto sobre quais os certificados a utilizar enquanto navega.

Ver certificados
Clique neste botão para ver os certificados armazenados, importar novos certificados, e guardar ou eliminar certificados antigos no &brandShortName;.

Listas de revogação
O &brandShortName; pode utilizar listas de revogação de certificados (também conhecidas por CRLs - Certificate Revocation Lists ou Lista de Certificados a Revogar) para garantir que os seus certificados não são inválidos. Prima o botão listas de revogação para gerir as CRLs instaladas no seu computador.

Verificação
O &brandShortName; pode verificar a validade dos seus certificados utilizando o OCSP (Online Certificate Status Protocol) sempre que são visualizados ou utilizados. O &brandShortName; não utiliza o OCSP por omissão, mas se quiser activar esta opção, pode fazê-lo aqui. Só deve necessitar de alterar esta opção, se o seu acesso à Internet o exigir.

Dispositivos de segurança
Os dispositivos de segurança podem cifrar e decifrar ligações e armazenar certificados e senhas. Se necessitar de utilizar um outro dispositivo de segurança para além daquele que o &brandShortName; utiliza, prima o botão Dispositivos de segurança.
