Controlar os Popups

Este documento explica todas as &pref.plural; disponíveis no &brandFullName; relativas ao controlo de popups.

Nesta secção:
  • O que são Popups?
  • &pref.pluralCaps; para Bloquear Popups

O que são Popups?

As janelas de Popup, ou popups, são janelas que surgem automaticamente sem a sua permissão. Elas podem variar de tamanho, mas geralmente não ocupam o ecrã inteiro. Alguns popups abrem no topo da janela actual do &brandShortName;, enquanto outras surgem por trás do &brandShortName; (popunders).

O &brandShortName; permite-lhe controlar os popups e os popunders através do painel de Conteúdos nas &pref.pluralCaps;. A opção de Bloquear Popups está activada por omissão, para não se preocupar em activá-la evitando que os popups surjam no &brandShortName;.

Ao bloquear um popup, o &brandShortName; apresenta uma barra de informação, bem como um ícone na barra de estado. Quando clicar no botão de &pref.pluralCaps; na barra de informações ou no ícone na barra de estado, surge um menu com as seguintes opções:

  • Permitir/Bloquear popups para este sítio
  • Edit &pref.pluralCaps; para Bloquear Popups
  • Não mostrar esta mensagem (info message) quando os popups são bloqueados
  • (mostrar um popup bloqueado)

Bloquear popups pode interferir com alguns sítios web: Alguns sítios web, incluindo alguns sítios de bancos, utilizam popups para funcionalidades importantes. Ao bloquear todos os popups desactiva essas funcionalidades. Para permitir que sítios web específicos utilizem popups, bloqueando os restantes, pode adicionar sítios web específicos à lista de sítios autorizados.

Bloquear popups nem sempre funciona: Embora o &brandShortName; bloqueie a maioria dos popups, alguns sítios web podem apresentar popups utilizando métodos alternativos, mesmo quando bloqueados.

&pref.pluralCaps; para Bloquear Popups

As &pref.pluralCaps; para Bloquear Popups estão localizadas no painel de Conteúdos do &pref.menuPath;.

A partir daí pode executar as seguintes acções:

  • Bloquear Janelas de Popup: Desmarque esta &pref.singular; para desactivar o bloqueio de todas as janelas de popups.
  • Diálogo de Excepções: Esta é a lista de sítios em que deseja permitir os popups. O diálogo tem as seguintes opções:
    • Permitir: Clique nesta opção para adicionar um sítio web à lista de excepções.
    • Remover Sítio: Clique nesta opção para remover um sítio web da lista de excepções.
    • Remover Todos os Sítios: Clique nesta opção para remover todos os sítios web da lista de excepções.

Nota: Bloquear popups nem sempre funciona e pode interferir com alguns sítios web. Para mais informações relacionadas com o bloqueio de popups, veja O que são Popups.
