Funcionalidades de acessibilidade do &brandFullName;

O &brandShortName; inclui várias funcionalidades para tornar o navegador e os conteúdos web acessíveis a todos os utilizadores, incluindo aqueles que têm a sua visão diminuída, são invisuais ou têm uma capacidade limitada para usar o teclado ou o rato.

Utilizar o teclado

Atalhos de teclado

Uma lista de atalhos de teclado comuns está disponível em Atalhos de teclado do &brandFullName;. Em modo ligado, pode encontrar um quadro de comparação dos atalhos de teclado implementados no &brandShortName;, Microsoft Internet Explorer e Opera.

Utilizar sempre as teclas do cursor para navegar nas páginas

Utilizar sempre as teclas do cursor para navegar nas páginas possibilita-lhe movimentar-se pelos conteúdos web como se estivesse dentro de um editor só de leitura. Pode usar o teclado para seleccionar conteúdos web e o copiar para a área de transferência. Pode activar esta funcionalidade seleccionando &pref.menuPath;, abrindo o separador Geral do painel Avançado e marcando Permitir que o texto possa ser seleccionado com o teclado &pref.singular;.

Pode premir F7 em qualquer altura para activar ou desactivar esta funcionalidade. Quando premir F7, o &brandShortName; irá pedir para confirmar se pretende mesmo activar esta funcionalidade. Pode desactivar esta pergunta marcando Não mostrar este diálogo novamente.

Procurar ao escrever

Procurar ao escrever permite navegar rapidamente para no texto ou para ligações numa página web. Esta funcionalidade tem dois modos. Pressione / e comece a escrever para localizar em todo o texto da página corrente ou pressione ' no início da frase para procurar apenas em ligações. O &brandShortName; irá destacar o texto ou ligações que correspondam à frase entretanto escrita.

Os seguintes atalhos controlam esta funcionalidade:

  • Escreva mais caracteres para afunilar a pesquisa. O &brandShortName; irá destacar a próxima ligação ou texto igual ou irá alertá-lo de que nada foi encontrado.
  • Pressione &accelKey;+G ou F3 para encontrar a próxima ocorrência da frase procurada.
  • Pressione &accelKey;+&shiftKey;+G ou &shiftKey;+F3 para encontrar a ocorrência anterior da frase procurada.
  • Pressione Backspace para desfazer a escrita do último caracter. O &brandShortName; retornará ou texto previamente destacado.
  • Pressione Esc para cancelar a localização. Pode também cancelar a pesquisa alterando o foco, deslizando a página com o rato ou teclado ou simplesmente esperar pelo temporizador.

Pode alterar o comportamento por omissão destas funcionalidades seleccionando &pref.menuPath;, abrindo o separador Geral do painel Avançado e marcando Procurar ao escrever &pref.singular;. Se esta opção estiver marcada, sempre que começar a escrever irá automaticamente começar a pesquisa em linha. Por omissão a pesquisa automática procura em todo o texto da página. Pode explicitamente premir ' ou / para começar num modo ou no outro.

Pode usar esta funcionalidade para encontrar botões, caixas de texto e outros controlos de formulários. Procure o texto imediatamente antes do controlo do formulário que pretende e pressione Tab. Ao premir Tab automaticamente sai da procura e coloca o foco no próximo campo do formulário ou botão depois de destacar o texto.

Tipos de letra e cores

Sobrepor os tipos de letra da página

Alguns sítios podem usar tipos de letra que são difíceis de ler e outros sítios podem estar baseados nos tipos de letra por omissão do navegador. Pode fixar os seus tipos de letra por omissão em &pref.menuPath;, seleccionando o painel Conteúdos e seleccionando os tipos por omissão na secção Tipos de letra & cores.

Pode especificar tipos por omissão para diferentes estilos, incluindo serifada, sans-serif e monoespaço clicando no botão Avançado.... No entanto, muitas páginas web especificam os seus próprios tipos de letra, por isso as suas escolhas podem não ter efeito nestas páginas. Para sobrepor os tipos de letra definidos nestas páginas, desmarque a &pref.singular; Permitir às páginas a escolha do seu tipo de letra em vez das escolhas acima.

Note que algumas páginas web podem ter uma visualização incorrecta dependendo das suas escolhas para o tipo de letra por omissão.

Fixar o tamanho mínimo dos tipos de letra

Alguns sítios web podem mostrar texto que é muito pequeno para poder ser lido confortavelmente. Para tornar essas páginas mais fáceis de ler, pode fixar o tamanho mínimo para os tipos de letra. Nas páginas web que utilizarem texto mais pequeno que esse tamanho, o &brandShortName; irá aumentar o texto para o tamanho mínimo definido por si.

Para definir o tamanho mínimo para os tipos de letra, vá a &pref.menuPath;, seleccione o painel Conteúdos e clique no botão Avançado... na secção Tipos de letra & cores. Pode especificar um tamanho mínimo para os tipos de letra nos menus suspensos.

Note que algumas páginas web podem ser visualizadas incorrectamente dependendo das suas escolhas para o tamanho mínimo do tipo de letra.

Redimensionar o texto da página

Pode temporariamente alterar o tamanho do texto em qualquer página web no menu Ver > Tamanho do texto. O novo tamanho ir-se-á manter até fechar a janela ou separador do navegador, mesmo que navegue para um sítio web diferente.

Os seguintes atalhos de teclado controlam o tamanho do texto:

  • Pressione &accelKey;++ para aumentar o tamanho do texto. Isto ampliará o texto para 120% do seu tamanho original, seguidamente para 150%, 200%, 300%, 450%, 675%, 1013%, e 1519%.
  • Pressione &accelKey;+- para diminuir o tamanho do texto. Isto reduzirá o texto para 90% do seu tamanho original, seguidamente para 50%, 33%, 22%, 15%, 10%, 7%, 4%, 3%, 2%, e 1%.
  • Pressione &accelKey;+0 para reinicializar o tamanho do texto para o tamanho por omissão da página web.

Note que algumas páginas web podem ser visualizadas incorrectamente se aumentar ou reduzir o tamanho do texto.

Sobrepor as cores da página

Alguns sítios web podem usar cores para o texto e de fundo da página que são difíceis de ler e outros sítios podem-se basear nas cores do navegador por omissão. Pode fixar as cores por omissão em &pref.menuPath;, seleccionando o painel Conteúdos e clicando no botão Cores na secção de Tipos de letra & cores. Pode também fixar as cores por omissão para o texto e para o fundo da página, bem como as cores por omissão para as ligações visitados e por visitar. Pode também especificar se as ligações devem ou não ser sublinhadas.

No entanto, muitas páginas web fixam as suas próprias cores, por isso as suas escolhas podem não ser visíveis. Para sobrepor as cores definidas pelas páginas web, desmarque a &pref.singular; Permitir às páginas escolher as suas próprias cores, em vez das acima seleccionadas.

Usar temas de alto contraste

O &brandShortName; detecta automaticamente se está a usar um tema de alto contraste e visualiza todo o conteúdo usando o esquema de cores de alto contraste escolhido. Isto ir-se-á sobrepor a todas as preferências do navegador ou das páginas web e afectará a interface do &brandShortName; propriamente dita (todos os menus, janelas e caixas de diálogo) e o conteúdo de qualquer página web que tenha visitado.

Para usar um tema de alto contraste, vá ao menu Iniciar > Painel de controlo > Opções de acessibilidade > Visualização e marque a &pref.singular; Utilizar 'Alto contraste'.

Controlar os conteúdos web

Bloquear janelas popup

Veja Controlar popups para informações em como bloquear janelas popup.

Desligar applets Java

Algumas páginas web providenciam uma experiência interactiva rica em applets Java™. No entanto, alguns utilizadores que apenas usem o teclado para navegar, poderão ter alguns problemas com alguns applets Java que automaticamente fixam o foco e não providenciam qualquer mecanismo para sair da applet e navegar para o resto da página web. Se isto é um problema para si, pode desactivar o Java em &pref.menuPath;, seleccionando Conteúdos e desmarcando a &pref.singular; Activar Java.

Restringir o comportamento do JavaScript

Pode desligar o JavaScript completamente nas &pref.menuPath;, seleccionando o painel de Conteúdos e desmarcando Activar JavaScript. Algumas páginas web depende do JavaScript e podem não funcionar correctamente com o JavaScript inibido.

Se quiser inibir o JavaScript completamente, existem vários parâmetros avançados para o JavaScript for forma a restringir o comportamento dos scripts. Imediatamente a seguir à caixa de verificação Activar JavaScript, clique o botão Avançado... para abrir a janela Definições avançadas de JavaScript. Pode controlar se é permitido aos scripts mover ou redimensionar janelas existentes, subir ou baixar janelas, desactivar ou substituir menus de contexto, esconder a barra de estado ou alterar o texto da barra de estado.
