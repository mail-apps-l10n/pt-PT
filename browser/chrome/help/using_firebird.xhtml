Utilizar o &brandFullName;

Bem-vindo ao &brandFullName;! O &brandShortName; é um navegador para a Internet que pode utilizar para visitar páginas web e pesquisar na Web.

Nesta secção:
  • Navegar em páginas Web
  • Pesquisar
  • Copiar, gravar e imprimir páginas
  • Melhorar a velocidade e eficiência

Navegar em páginas Web

Ver a sua página inicial

Quando inicia o &brandShortName;, verá a sua página inicial. Por omissão, verá a página do &brandShortName;.

Dicas:

  • Para mostrar mais conteúdo web no ecrã, pode usar o modo de Ecrã inteiro. O modo de Ecrã inteiro reduz as barras de ferramentas do &brandShortName; a uma pequena barra de ferramentas. Para activar o modo de Ecrã inteiro, seleccione Ver > Ecrã inteiro ou prima F11.
  • Para ir para a sua página inicial rapidamente, prima &altKey;+Home.

Mudar para outra página

Pode mudar para outra página web escrevendo o endereço ou URL na Barra de navegação. As URL's começam geralmente por "http://" seguidas de um ou mais nomes que identificam o endereço. Exemplo: "http://www.mozilla.org/".

  1. Clique na Barra de navegação para seleccionar a URL actual.
  2. Escreva a URL da página que quer visitar. A URL que escrever irá substituir o texto existente na Barra de navegação.
  3. Prima &enterKey;.

Dica: Para seleccionar a URL da Barra de navegação de forma rápida, prima &accelKey;+L.

Não sabe uma URL? Experimente escrever algo relativo à página que pretende visitar, como por exemplo, um nome, na Barra de navegação e prima &enterKey;. Surgirá então o primeiro tópico resultante de uma pesquisa no Google referente a esse termo.

Clicar numa ligação

A maioria das páginas web contem ligações onde pode clicar e visitar outras páginas.

  1. Mova o ponteiro do rato até se converter numa mão a apontar. Isto acontece sempre que o ponteiro do rato estiver sobre uma ligação. A maioria das ligações está sob a forma de texto sublinhado mas os botões e as imagens podem ser igualmente ligações.
  2. Clique uma vez na ligação. Enquanto a rede localiza a página referente à ligação, surgirão mensagens do seu estado na parte inferior da janela.

Seguir os seus passos

Há várias formas de revisitar páginas:

  • Para voltar atrás ou avançar numa página, clique nos botões Retroceder ou Avançar.
  • Para voltar atrás ou avançar mais do que uma página, clique nos pequenos triângulos existentes nos botões Retroceder e Avançar. Verá uma lista de páginas que visitou recentemente; Para voltar uma página, escolha-a da lista.
  • Para ver a lista de todos as URL's que escreveu na Barra de navegação, clique na seta à direita da Barra de navegação. Para ver uma página, seleccione-a da lista.


  • Para escolher páginas que visitou nesta sessão, abra o menu Histórico e utilize a lista da secção inferior do menu.
  • Para escolher páginas que visitou em várias sessões, abra o menu Histórico e seleccione Mostrar na barra lateral. Verá a Barra do Histórico. A Barra do Histórico mostra a lista de pastas. Ao clicar nas pastas, surgem subpastas ou os títulos das páginas web. Para ver uma página, clique no título dessa página.

Parar e recarregar

Se a página estiver a carregar lentamente ou se desistir de visitar uma página, clique no botão Parar.

Para recarregar a página actual ou para obter uma versão mais actualizada da página, clique no botão Recarregar ou prima &accelKey;+R.

Navegar com separadores

Para visitar várias páginas ao mesmo tempo, pode usar a Navegação com Separadores para navegar na internet de forma mais rápida e fácil.

A Navegação em Separadores permite abrir separadores; cada uma pode conter uma página web, dentro de uma janela &brandShortName;. Não é obrigatório ter várias janelas abertas para visitar várias páginas web diferentes. No entanto, estas páginas ocupam espaço no seu ambiente de trabalho. Pode abrir, fechar e recarregar páginas web, convenientemente, uma em cada sítio sem ter de mudar de janela.

Para mais informações, visite o tópico Navegar com Separadores.

Utilizar a barra lateral

A Barra Lateral é a área situada do lado esquerdo do ecrã que pode utilizar para ver marcadores ou o histórico. Existem também extensões que possibilitam novas formas de utilização da Barra Lateral.

Para ver um item da Barra Lateral, seleccione Ver > Barra Lateral. A partir daí pode escolher o separador da Barra Lateral que quiser.

Pesquisar

Pesquisar na Web

Pesquisar páginas na web ou sobre um tópico específico é simples, basta escrever algumas palavras na Barra de Pesquisa do &brandShortName;:

Por exemplo, se quiser encontrar informações sobre baby dolls:

  1. Clique na Barra de Pesquisa.
  2. Escreva a frase baby doll. Tudo o que escrever substituirá o texto existente na barra de pesquisa.
  3. Prima &enterKey; para pesquisar.

Os resultados da pesquisa "baby doll" surgirão na janela do &brandShortName;.

Seleccionar o motor de pesquisa

Pode escolher outro motor de pesquisa, clicando no ícone respectivo e seleccionando-o. Alguns motores de pesquisa, como o Google, pesquisam em toda a web; outros, como o Amazon.com, apenas procuram em sítios específicos.

Gerir motores de pesquisa

Clique no ícone do motor de pesquisa e seleccione Gerir motores de pesquisa... para adicionar, reordenar ou remover motores de pesquisa. Seleccione um motor de pesquisa e clique no botão apropriado para o mover dentro da lista ou para o remover. Pode instalar motores de pesquisa adicionais a partir dos Extras do &brandShortName;.

Pesquisar palavras seleccionadas numa página Web

O &brandShortName; permite a pesquisa por palavras seleccionadas dentro de uma página web:

  1. Seleccione (realce) quaisquer palavras numa página web.
  2. Clique com o botão do lado direito do rato Prima &ctrlKey;, clique no botão do rato, e escolha Pesquisar [motor de pesquisa] por "[as suas palavras seleccionadas]" do menu.

O &brandShortName; abrirá um novo separador e utilizará o seu motor de pesquisa por omissão para pesquisar as suas palavras seleccionadas.

Pesquisar dentro de uma página

Para encontrar um texto específico na sua página actual do &brandShortName;:

  1. li>Prima CtrlCmd+F ou seleccione Editar > Procurar nesta Página para abrir a barra de pesquisa na parte inferior do seu navegador.
  2. Escreva o texto que pretende encontrar. A pesquisa inicia automaticamente assim que começa a escrever no campo de pesquisa
  3. A barra de pesquisa possibilita as seguintes opções:
    • Seguinte: pesquisa na página, entre o texto que se encontra abaixo da posição actual do cursor.
    • Anterior: pesquisa na página, entre o texto que se encontra acima da posição actual do cursor.
    • Marcar todas: realça os resultados da sua pesquisa na página actual.
    • Sensível a maiúsculas/minúsculas: limita a pesquisa ao texto que tenha a mesma capitalização.

Para encontrar a mesma palavra ou frase de novo, prima F3 ou seleccione Editar > Procurar novamente.

Dica: Marque a opçãopreferência Verificar a minha ortografia durante a escrita no separador Geral do painel Avançado das Ferramentas > OpçõesEditar > PreferênciasPreferências do Firefox para activar o modo de Pesquisa Rápida na barra de Pesquisa. Ao ser activada, a barra da Pesquisa Rápida surge automaticamente e começa a pesquisar assim que começar a escrever. Ao contrário da barra de pesquisa, a barra da Pesquisa Rápida fecha-se automaticamente alguns segundos depois se não for utilizada.

Copiar, gravar e imprimir páginas

Copiar parte de uma página

Para copiar o texto de uma página:

  1. Seleccione o texto.
  2. Seleccione Editar > Copiar da Barra do Menu.

Pode colar o texto noutros programas.

Para copiar uma ligação (URL) ou uma ligação de uma imagem de uma página:

  1. Coloque o ponteiro sobre a ligação ou imagem.
  2. Clique no botão direito do rato Prima &ctrlKey; e clique na ligação ou imagem para que o menu surja.
  3. Escolha Copiar endereço da ligação ou Copiar endereço da imagem. Se uma imagem for também uma ligação, pode seleccionar qualquer das duas opções.

Pode colar a ligação noutros programas ou na barra de navegação do &brandShortName;.

Guardar uma página

Para guardar uma página inteira do &brandShortName;:

  1. Seleccione Ficheiro > Guardar Página Como.... Verá uma a janela de configuração.
  2. Escolha o local onde deseja guardar a página.
  3. Escolha o formato em que deseja guardar a página:
    • Página Web, completa: Guarda toda a página web, incluindo as imagens. Esta opção permite visualizar a página como é originalmente mostrada com imagens, mas pode não manter a estrutura de ligações HTML da página original. O &brandShortName; cria uma nova directoria onde a página é guardada para guardar as imagens e outros ficheiros necessários para a visualização da página web integral.
    • Página Web, apenas HTML: Guarda a página original sem as imagens. Esta opção preserva a estrutura de ligações HTML num ficheiro.
    • Ficheiro de Texto: Guarda a página original como um ficheiro de texto. Esta opção não preserva a estrutura de ligações HTML original, mas permite-lhe ver a versão de texto da página web em qualquer editor de texto.
  4. Atribua um nome de ficheiro para a página e clique em Guardar.

Para guardar uma frame de uma página web:

  1. Posicione o ponteiro do rato na frame.
  2. Prima &ctrlKey; e clique naClique com o botão direito do rato sobre a frame para surgir o menu.
  3. Seleccione a opção Esta frame > Guardar frame como... do submenu. Verá uma janela de configuração.
  4. Escolha o local onde deseja guardar a página.
  5. Escolha o formato em que deseja guardar a página.
  6. Atribua um nome de ficheiro para a página e clique em Guardar.

Guardar um ficheiro no seu disco permite-lhe a possibilidade de visualizar a página mesmo quando não está ligado à Internet.

Para guardar uma imagem de uma página:

  1. Posicione o ponteiro do rato sobre a imagem.
  2. Clique com o botão direito do rato sobre aPrima &ctrlKey; e clique na imagem para surgir o menu.
  3. Seleccione a opção Guardar imagem como.... Verá uma janela de configuração.
  4. Escolha o local onde deseja guardar a imagem.
  5. Atribua um nome à imagem e clique em Guardar.

Para guardar uma página sem a mostrar (o que é útil na pesquisa de uma página não formatada, e que não pretende ver):

  1. Posicione o ponteiro do rato sobre uma ligação da página.
  2. Clique com o botão direito do rato sobre aPrima &ctrlKey; e clique na ligação para surgir o menu.
  3. Seleccione a opção Guardar ligação como.... Verá uma janela de configuração.
  4. Escolha o local onde deseja guardar a página.
  5. Atribua um nome de ficheiro para a página e clique em Guardar.

Importante: Algumas ligações, ao serem clicadas, descarregam e guardam ficheiros automaticamente no seu disco. As URL's destas ligações começam geralmente por "ftp" ou terminam com uma extensão de ficheiro, como por exemplo "au" ou "mpeg". Estas ligações podem estar associadas a ficheiros de software, som, ou vídeo e podem abrir aplicações que suportem esses ficheiros.

Dica: Para colocar uma imagem de fundo no seu Ambiente de Trabalho, clique com o botão direito do ratoprima &ctrlKey;, clique no botão do rato sobre a imagem e escolha Definir como fundo do ambiente de trabalho... a partir do menu.

Imprimir uma página

Para imprimir a página actual:

  • Seleccione a opção Ficheiro > Imprimir.

Seleccione o texto a imprimir:

  • Seleccione o texto a imprimir na página actual.
  • Seleccione a opção Ficheiro > Imprimir. Surgirá a janela de configuração de impressão.
  • Em Área de impressão, clique em Selecção.

O autor da página web e o tamanho da folha de impressão determinam o posicionamento do conteúdo na página a imprimir, e não o tamanho da janela no ecrã. O texto é moldado e as imagens reposicionadas para se adaptarem ao tamanho da folha.

Pré-visualizar impressão

Para ter uma ideia de como a página vai ficar depois de impressa, pode utilizar a Pré-Visualização:

  • Seleccione a opção Ficheiro > Pré-visualizar impressão.

Em Pré-visualizar impressão, pode executar várias acções com as páginas que quer imprimir:

  • Pré-visualizar cada página individualmente: Clique nos botões Seguinte , Anterior , Primeira , ou Última para navegar entre as páginas.
  • Alterar a escala (dimensão) das páginas: No menu Escala, escolha "50%" para que cada página passe a ter metade do tamanho original da página. Para redimensionar páginas automaticamente para que não ultrapassem a largura da página impressa, escolha "Redimensionar para caber na largura da página". Pode definir a sua própria percentagem escolhendo "Personalizar...".
  • Alterar a orientação da página: Escolha Retrato para posicionar a página com o lado menor para cima. Escolha Paisagem para posicionar a página de lado, com o maior lado para cima.
  • Configurar Página: Clique em Configurar página para definir opções avançadas das páginas que pretende imprimir.
  • Imprimir: Clique em Imprimir para imprimir as páginas.

Nota: Algumas funções da Pré-visualização de impressão podem ser diferentes ou estar indisponíveis para Mac OS e Linux.

Configurar página

Nota: Algumas funções da Configurar página podem ser diferentes ou estar indisponíveis para Mac OS e Linux.

Para configurar a forma como as páginas são impressas no &brandShortName;, pode utilizar a opção Configurar página:

  • Seleccione a opção Ficheiro > Configurar página.

Na opção Configurar página, pode executar várias acções com as páginas que pretende imprimir:

  • Formato e opções: Escolha a orientação, escala e outras opções:
    • Orientação:
      • Retrato: Escolha esta opção para posicionar a página com o lado menor para cima.
      • Paisagem: Escolha esta opção para posicionar a página de lado, com o lado maior para cima.
    • Escala: Defina a percentagem do tamanho original. Por exemplo, escreva "50" para que cada página passe a ter metade do tamanho original da página.
      • Redimensionar para caber na largura da página: Seleccione esta opção para redimensionar a página automaticamente à largura do papel.
    • Opções:
      • Imprimir fundo (cores e imagens): Seleccione esta opção para imprimir as imagens e as cores de fundo. Se esta opção não for seleccionada, serão impressas apenas as imagens e cores que surjam à frente.
  • Margens e cabeçalho/rodapé: Clique neste separador para definir as margens, cabeçalhos e rodapés:
    • Margens:
      • Topo, Fundo, Esquerda, Direita: Defina os valores nas unidades fornecidas para as margens de topo, fundo, esquerda e direita.
    • Cabeçalhos e rodapés: Cada lista em menu representa a área do cabeçalho ou do rodapé. A primeira linha de listas é referente ao lado esquerdo, central, e direito dos cabeçalhos. A linha de baixo refere-se ao lado esquerdo, central e direito dos rodapés. Nas listas, escolha uma das seguintes opções:
      • --em branco--: Não mostra nada nesta área.
      • Título: Mostra o título da página web.
      • URL: Mostra a URL da página web (geralmente as URLs começam com "http://").
      • Data/Hora: Mostra a data e hora de impressão da página web.
      • Página #: Mostra o número da página de cada página.
      • Página # de #: Mostra o número da página actual num total de páginas. Por exemplo, se imprimir cinco páginas, a terceira página seria representada assim: "3 de 5".
      • Personalizar: Defina o seu próprio texto. Pode incluir qualquer um dos seguintes códigos para imprimir informação específica:
        • &PT: Numeração da Página com o Total (Exemplo: "3 de 5")
        • &P: Número da Página
        • &D: Data
        • &U: URL
        • &T: Título

Dica: Para pré-visualizar as alterações da opção Configurar página, utilize a Pré-visualização de impressão.

Melhorar a velocidade e eficiência

Gerir diferentes tipos de ficheiros

O &brandShortName; consegue ler vários tipos de ficheiros. No entanto, o &brandShortName; necessita de Plugins ou aplicações externas para ler alguns ficheiros como os de vídeo ou áudio. Se o &brandShortName; não tiver o Plugin necessário ou se não souber que aplicação usar para abrir o ficheiro, continua a ser possível guardar o ficheiro no seu computador. Ao guardar ficheiros, pode geri-los através do Gestor de Transferências.

Os Plugins aumentam as funcionalidades do &brandShortName; e são executados através deste. Plugins como Sun Java, Macromedia Flash, e RealNetworks RealPlayer permitem, ao &brandShortName;, a leitura de ficheiros multimédia e a execução de pequenas aplicações como por exemplo, filmes, animações e jogos. Pode encontrar os Plugins mais comuns na Página de Plugins na Mozilla Update.

Se não conseguir abrir ficheiros com o &brandShortName;, pode optar por abri-los com aplicações externas ao &brandShortName;. Por exemplo, pode utilizar o media player da sua preferência para ouvir ficheiros de MP3.

Se o &brandShortName; não souber como lidar com um ficheiro, pode especificá-lo na janela Acções Pós-Transferências. Para abrir esta janela, seleccione &pref.menuPath;, abra o painel Conteúdo, e clique no botão Gerir na secção Tipos de ficheiro. Pode alterar a acção para um tipo de ficheiro ou removê-la. Para mais informações sobre esta janela de configuração visite a secção Tipos de ficheiro.

Se ao clicar num ficheiro, o &brandShortName; não souber como lidar com ele, surgirá uma janela de configuração para Abrir o ficheiro, mostrando as seguintes opções:

  • Abrir com: O &brandShortName; abre o ficheiro com o programa definido por omissão nas suas definições de sistema. Pode escolher um outro programa seleccionando Outro... entre as opções da lista.
  • Guardar no disco: O &brandShortName; permite-lhe escolher um local onde deseja guardar o ficheiro. Se seleccionou Guardar ficheiros no ... no painel Transferências das &pref.pluralCaps;, o ficheiro será guardado automaticamente na pasta definida por omissão.
  • Fazer isto automaticamente para ficheiros deste tipo de agora em diante: Quando esta opção é seleccionada, o &brandShortName; processa a acção seleccionada automaticamente, na próxima vez que se fizer uma transferência de um ficheiro deste tipo.

Dica: Pode limpar as suas &pref.plural; gravadas na secção de Executar automaticamente a acção associada a cada um dos seguintes tipos de ficheiro na janela de configuração Acções Pós-Transferência.

Definir o &brandShortName; como o seu navegador por omissão

O &brandShortName; mostra páginas web na Internet e no seu computador. Para abrir páginas web de forma simples, pode definir o &brandShortName; como o seu navegador por omissão.

Para definir o &brandShortName; como seu navegador por omissão, seleccione &pref.menuPath; e clique no botão Verificar Agora no painel Geral.

Definir &pref.plural; de cache

O &brandShortName; armazena cópias das páginas visitadas frequentemente na cache. Ao fazê-lo, o &brandShortName; não precisa de buscar a página à internet de cada vez que visitar essa página web.

Para definir o tamanho da cache:

  1. Seleccione &pref.menuPath;.
  2. Abra o painel Avançado e clique no separador Rede.
  3. Insira um valor no campo Utilizar até para definir o tamanho da cache. O tamanho por omissão é 50MB.

Importante: Uma cache maior permite que mais dados sejam rapidamente encontrados, embora ocupe mais espaço do seu disco rígido.

Ao fechar o &brandShortName;, este faz a manutenção da cache. Se a manutenção demorar mais do que o que deseja, tente reduzir o tamanho da cache do disco.

Para recarregar uma página a qualquer altura, clique no botão Recarregar na Barra de navegação do &brandShortName;. O &brandShortName; carrega e apresenta a última versão da página.
