Glossário

Autenticação
A utilização de uma senha, certificado, número de identificação pessoal (PIN), ou outra informação para validar uma identidade sobre uma rede de computadores.
Barra de endereço
O campo (e respectivos botões) junto ao topo da janela do &brandShortName; onde pode escrever uma URL ou termos de pesquisa.
Barra de estado
A barra de ferramentas que aparece no fundo de qualquer janela do &brandShortName;. Esta mostra ícones de estado à direita.
Barra de ferramentas dos Marcadores
A barra de ferramentas personalizada que aparece logo abaixo à barra de endereço por omissão no &brandShortName;. Contém botões para os seus marcadores favoritos (ou pastas que contenham grupos de marcadores) que pode adicionar ou remover.
Barra de navegação
A barra de ferramentas próxima do topo da janela do &brandShortName; que inclui os botões Retroceder e Avançar.
Cache
Uma colecção de cópias de páginas web armazenadas no disco rígido ou na memória RAM do seu computador. O &brandShortName; armazena estas cópias enquanto navega na Internet. Quando clica numa ligação ou escreve uma URL para aceder a uma página web em particular da qual já exista uma cópia na cache, o &brandShortName; compara a cópia armazenada com o original. Se não existirem diferenças, o &brandShortName; utiliza a cópia armazenada em vez de transferir novamente o original, economizando tempo na transferência e no processamento.
Certificado
O equivalente digital a um cartão de identificação. Um certificado específica o nome de um indivíduo, empresa, ou outra entidade e certifica uma chave pública, que é incluída no certificado, que pertence a essa entidade. Quando assina digitalmente uma mensagem ou outros dados, a assinatura digital para essa mensagem é criada em conjunto com a chave privada que corresponde à chave pública do seu certificado.
Chave privada
Uma de um par de chaves utilizadas na criptografia de chave pública. A chave privada é mantida em segredo e é utilizada para decifrar dados que foram cifrados com a chave pública correspondente.
Cifragem
O processo de baralhar informação de forma a disfarçar o seu significado. Por exemplo, ligações cifradas entre computadores fazem com que seja muito difícil a terceiros ordenar e decifrar a informação que circula na ligação. Informação cifrada só pode ser decifrada por quem tenha a chave apropriada.
Cliente
Aplicação (como um navegador de Internet) que envia pedidos para e recebe informação de um servidor, que geralmente está a ser executado em outro computador. Um computador no qual a aplicação de cliente é executada é também designado por cliente.
Cookie
Um pequeno pedaço de informação armazenado no seu computador por alguns sítios. Quando visita um sítio deste género, o sítio pede ao &brandShortName; para armazenar uma ou mais cookies no disco rígido do seu computador. Mais tarde, quando regressa ao sítio, o &brandShortName; envia para o sítio as cookies que lhe pertencem. As cookies ajudam os sítios web a manter informação sobre si, tais como os conteúdos do seu carrinho de compras. Pode definir opções de cookies para controlar como as cookies são utilizadas e que quantidade de informação está disposto a permitir que os sítios armazenem nas mesmas.
Criptografia
A arte e prática de baralhar (cifrar) e ordenar (decifrar) informação. Por exemplo, são utilizadas técnicas criptográficas para baralhar e ordenar informação entre sítios web comerciais e o &brandShortName;.
Decifragem
O processo de ordenar dados que foram cifrados. Ver também cifragem.
Extensible Markup Language (XML)
Uma norma aberta para descrever dados. Ao contrário do HTML, o XML permite ao programador de uma página web a definição de tags especiais. Para mais informações, aceda ao documento em linha do W3C Extensible Markup Language (XML).
FIPS PUBS 140-1
A Federal Information Processing Standards Publications (FIPS PUBS) 140-1 é uma norma governamental americana para implementação de módulos de criptografia — hardware ou software que cifra e decifra dados ou executa outras operações criptográficas (como a criação e verificação de assinaturas digitais). Muitos produtos vendidos ao governo americano têm de estar de acordo com uma ou mais normas FIPS.
Fonte
Uma página XML que contém uma lista de ligações para outras páginas web. Programas especiais podem ler estas fontes e criar uma lista de títulos a partir das ligações, actualizando automaticamente a lista quando esta é alterada. Sítios Web de notícias utilizam as fontes para publicarem rapidamente as últimas novidades, e jornais pessoais online utilizam frequentemente fontes para notificar rapidamente os visitantes sobre as novas entradas. Ver também Marcador dinâmico.
Gestor de senhas
Parte do &brandShortName; que pode ajudá-lo a lembrar de alguns ou de todos dos seus nomes e senhas armazenando-os no disco rígido do seu computador e introduzindo-os automaticamente quando visita estes sítios.
Hypertext Markup Language (HTML)
O formato de documento utilizado para as páginas web. A normal HTML define tags, ou código, utilizado para definir a disposição do texto, tipos de letra, estilos, imagens, e outros elementos que formam uma página web.
Identificação digital
Ver certificado.
Internet
Uma rede mundial de milhões de computadores que comunicam entre si utilizando protocolos normalizados como o TCP/IP. Originalmente desenvolvido pelos militares norte-americanos em 1969, a Internet cresceu para incluir instituições de pesquisa e educação e, no final dos anos 90, milhões de empresas, organizações e indivíduos. Hoje a Internet é utilizada para correio electrónico, navegação na World Wide Web, mensagens instantâneas, listas de correio electrónico, e para muitos outros fins.
Java
Uma linguagem de programação desenvolvida pela Sun Microsystems. Uma aplicação simples de Java pode ser executada em muitos diferentes tipos de computadores, evitando a necessidade de os programadores criarem uma versão diferente para cada tipo de computador. Depois de o Java estar instalado, o &brandShortName; pode transferir e executar automaticamente programas Java (também chamados de applets).
JavaScript
Uma linguagem de programação normalmente utilizada para construir páginas web. Os programadores utilizam o JavaScript para tornar as páginas web mais interactivas; É comum o JavaScript ser utilizado para a validação dinâmica de formulários e selecção de botões. O JavaScript pode ser utilizado com o Java mas é, tecnicamente, uma linguagem separada. Não é necessário Java para o JavaScript funcionar correctamente.
Marcador
Um endereço de uma página web (URL) armazenado para que a possa visitar facilmente clicando no ícone do marcador na Barra de ferramentas dos Marcadores ou escolhendo o nome do marcador do menu Marcadores.
Marcadores dinâmicos
Um tipo especial de marcadores que actua como uma pasta que contém as ligações de uma fonte. Pode criar um marcador activo visitando um sítio com uma ou mais fontes, clicando no ícone do marcador dinâmico na barra de endereço, e seleccionando a fonte que deseja utilizar.
Motor de pesquisa
Um programa baseado na web que permite aos utilizadores procurar por ou obter informações específicas da Internet. O motor de pesquisa pode procurar o texto completo de documentos web ou uma lista de senhas; pode também recorrer a bibliotecários que revêem e indexam documentos web manualmente. Tipicamente, o utilizador escreve uma palavra ou uma frase numa caixa de pesquisa, e o motor de pesquisa mostra ligações para as páginas web relevantes.
Página inicial
A página que o &brandShortName; tem definida para ser visualizada sempre que o inicia ou quando pressiona o botão Página inicial. Este termo é também utilizado para se referir à página principal de um sítio web do qual pode explorar o resto do sítio.
Página web
Um documento simples da Internet que é especificado por um endereço ou URL único. Uma página web pode também conter texto, ligações e imagens.
PKCS #11
Norma criptográfica de chave pública que normaliza dispositivos de segurança tais como os smart cards.
Plugin
Os plugins adicionam novas capacidades ao &brandShortName;, tais como a capacidade de reproduzir clips de áudio ou de vídeo. Ao contrário de outras aplicações acessórias, um plugin instala-se no directório dos plugins dentro do directório principal de instalação do &brandShortName; e, tipicamente, pode ser aberto dentro do &brandShortName;. Por exemplo, um plugin de áudio permite-lhe ouvir ficheiros áudio numa página web ou numa mensagem de correio electrónico. O Macromedia Flash Player e o Java são exemplos de aplicações que podem funcionar como plugins.
Protocolo de endereços da Internet (endereço de IP)
Um endereço de um computador numa rede TCP/IP. Todos os computadores na Internet têm um endereço de IP. Os Clientes podem ter um endereço de IP fixo ou um que é atribuído dinamicamente sempre que o computador é ligado à rede.
Protocolo de Transferência de Ficheiros (FTP)
Uma norma que permite aos utilizadores transferir ficheiros de um computador para outro através de uma rede. Pode utilizar o &brandShortName; para descarregar ficheiros utilizando o FTP.
Proxy
Um programa intermediário que actua como um servidor e um cliente com o propósito de fazer pedidos em nome de outros clientes.
Secure Sockets Layer (SSL)
Um protocolo que permite uma autenticação mútua entre um cliente e um servidor com o objectivo de estabelecer uma ligação autenticada e cifrada. O SSL corre sobre o TCP/IP e abaixo do HTTP, LDAP, IMAP, NNTP, e outros protocolos de rede de alto nível. A nova norma do Internet Engineering Task Force (IETF) denominada por Transport Layer Security (TLS) é baseada no SSL. Ver também autenticação, cifragem.
Senha mestra
Uma senha utilizada para proteger as senhas guardadas e outras informações privadas. O &brandShortName; irá perguntar-lhe por esta senha mestra quando tentar aceder a estes dados. Se tiver múltiplos dispositivos de segurança, cada dispositivo de segurança necessitará de uma senha mestra separada.
Servidor
Software (como o software que serve páginas web) que recebe pedidos de e envia informação para um cliente, que habitualmente é executado num computador diferente. Um computador no qual o software de servidor é executado também é denominado por servidor.
Sítio seguro
Um sítio que utiliza cifragem nas ligações com o &brandShortName; para prevenir que utilizadores maliciosos da Internet visualizem os dados transmitidos. Quando visita sítios seguros, o &brandShortName; mostra o ícone de um cadeado na barra de estado e na barra de endereço. O &brandShortName; mostra também o nome do domínio do sítio na barra de estado (para prevenir que sítios maliciosos roubem os seus dados) e altera a cor de fundo da barra de endereço para amarelo.
Alguns sítios podem ser constituídos por dados seguros e não seguros; os dados não seguros são usualmente não privados, embora não seja garantido que assim seja. Para estes sítios o &brandShortName; mostra um ícone de um cadeado com uma barra sobre o mesmo na barra de estado e na barra de endereço e não mostrará o nome do domínio do sítio na barra de estado para que saiba que a página actual não é completamente segura.
Sítio web
Um grupo de páginas web relacionadas, ligadas através de ligações e geridas por uma empresa, organização ou indivíduo. Um sítio web pode incluir texto, imagens, ficheiros de áudio e de vídeo, e ligações para outros sítios web.
TLS
Ver Secure Sockets Layer (SSL).
Transmission Control Protocol/Internet Protocol (TCP/IP)
Um protocolo Unix utilizado para interligar computadores com diversos sistemas operativos. O TCP/IP é essencialmente um protocolo de Internet que se tornou numa norma global.
Uniform Resource Locator (URL)
O endereço normalizado que diz ao &brandShortName; como localizar um ficheiro ou outro recurso na Web (por exemplo, http://www.mozilla.org). Digite URLs na barra de endereço do &brandShortName; para aceder a páginas. As URLs são também utilizadas nas ligações das páginas web para ir para outras páginas. Também conhecidas como endereço web ou de Internet.
World Wide Web
Também conhecida com a Web. Uma parte da Internet que é constituída por páginas web armazenadas por servidores web e visualizadas por clientes denominados por navegador (como o &brandShortName;).
